mod Harbour V2

Harbour entra en la web. Bienvenidos

Descripción

mod Harbour es un mod para Apache de codigo abierto que permite llevar tus programas harbour a la web. Toda la funcionalidad que te ofrecen otros lenguajes están ahora disponibles para que puedas llevar de una manera eficiente y profesional a la web.

Mientras los lenguajes standards (html, javascript, css) seguiran siendo los que utilizemos para crear la parte frontend, Harbour será el lenguaje de programación a nivel backend como otros lenguajes conocidos: phyton php, … Entra en la web con la potencia y fiabilidad que siempre a ofrecido Harbour

La mejor manera de instalar modHarbour esta indicado en nuestra wiki
