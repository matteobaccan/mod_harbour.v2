mod Harbour V2

Harbour comes to the web. Welcome

Description

mod Harbour is an open source mod for Apache that allows you to bring your harbour programs to the web. All the functionality offered by other languages are now available so you can take in an efficient and professional way to the web.

While the standard languages (html, javascript, css) will continue to be the ones we use to create the frontend part, Harbour will be the programming language at the level backend like other known languages: phyton php, … Enter the web with the power and reliability that Harbour has always offered

The best way to install modHarbour is indicated in our wiki
